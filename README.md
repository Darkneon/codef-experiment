Codef-experiment
================

Having fun with CODEF

Instructions
================
To hear the music, the file must be loaded from a web server.

1. Open a terminal
2. Navigate to the folder with the source code
3. Type: python -m SimpleHTTPServer
4. Go to localhost:8000 in your browser
5. Enjoy!
